# Nomad List 詳細調査レポート

## 基本情報
- **URL**: https://nomadlist.com
- **創設者**: Pieter Levels (@levelsio)
- **ローンチ**: 2014年
- **収益（2024年）**: $5.3M以上
- **有料顧客数**: 29,000人以上

---

## 1. コンセプト・概要

### サービスの目的
Nomad Listは、デジタルノマド（リモートワークをしながら世界を旅する人々）が「住み・働くのに最適な都市」を見つけるためのプラットフォームです。

### 主な機能
- **都市ランキング**: 500以上の都市について、50,000以上のデータポイントを収集
- **評価基準**: インターネット速度、生活費、天候、安全性、税金、ビザ要件など
- **コミュニティ機能**: Slackベースのチャットコミュニティ
- **Nomad Soulmates**: リモートワーカー向けのマッチング機能
- **フォーラム・イベントリスティング**: デジタルノマド向けの情報交換の場

### 解決している課題
- 「低い生活費」「良い気候」「高速インターネット」などの条件を比較できる総合的な情報サイトの不在

---

## 2. ターゲットユーザー

### 想定ユーザー層
- **デジタルノマド**: ノートパソコン一つで旅をしながらリモートワークする人々
- **リモートワーカー**: 場所に縛られない働き方を模索している人
- **フリーランサー**: 生活費を抑えながら収入を最大化したい自営業者
- **スタートアップ創業者**: 低コストで起業を目指す人

### ユーザーの課題・ニーズ
- 次の滞在先の選定（気候、コスト、インフラ）
- 現地での作業場所の発見
- 同じライフスタイルを持つ仲間との出会い
- ビザ・税金などの実務情報へのアクセス

## 3. ビジネスモデル

### 収益モデル
**メンバーシップ制（買い切り型）**
- **通常価格**: $299.98（ライフタイムアクセス）
- **セール価格**: $99〜$149（定期的に50%オフセールを実施）
- 開始当初は$5から始まり、段階的に値上げ

### 推定収益
| 年 | 収益 |
|------|------|
| 2017年 | $120K |
| 2020年 | $310.2K |
| 2021年 | $505.1K |
| 2022年 | $700K |
| 2023年 | $704.2K |
| 2024年 | **$5.3M** |

## 4. 歴史・成長過程

### タイムライン
- **2013年4月**: Pieter Levels、オランダから世界を旅し始める
- **2014年**: 「12ヶ月で12スタートアップ」チャレンジを開始
- **誕生の経緯**: Twitterでスプレッドシートを公開 → 1,000人以上がコントリビュート → バイラル化 → 1ヶ月以内にMVPをローンチ
- **2014〜2017年**: Wired、BBC、CNN、Time、HuffPoなどメディアに取り上げられる
- **2024年**: 10万人以上の会員、29,000人以上の有料顧客を達成

### 重要なマイルストーン
- 外部資金調達: **$0**（完全ブートストラップ）
- 従業員: **0人**（ソロファウンダー）

## 5. マーケティング戦略

### Twitter/SNS戦略
- **Building in Public**の先駆者
- すべての開発過程、収益数字をTwitterで公開
- 自身のドメイン（levels.io）はTwitterプロフィールにリダイレクト

### コミュニティ構築
**3フェーズ戦略**:
1. 共通の関心事を持つ人々を集める
2. データを収集・可視化して価値を提供
3. コミュニティを活用して関連プロダクトを展開

### 成長チャネル
- **SEO**: 検索エンジン経由での自然流入
- **コンテンツマーケティング**: リモートワークに関するデータや記事
- **口コミ**: 広告費 $0、顧客獲得コスト $0

## 6. 技術スタック

| カテゴリ | 技術 |
|----------|------|
| バックエンド | Vanilla PHP |
| フロントエンド | jQuery |
| データベース | SQLite |

### 設計思想
- 各マイクロSaaSは基本的に単一の`index.php`ファイル
- フレームワークレス
