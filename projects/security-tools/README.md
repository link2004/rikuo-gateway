# セキュリティツールの世界へようこそ

あなたがこのページを開いたということは、「セキュリティ」という言葉の向こう側に何があるのか、知りたくなったということだ。

---

## ハッカーとは何者か

映画に出てくるハッカーは、暗い部屋でキーボードを叩き、緑色の文字が流れる画面を見つめている。あれは半分本当で、半分嘘だ。

本当のハッカーは、システムの「穴」を見つける人間だ。その穴を悪用する者もいれば、穴を塞ぐ者もいる。同じ知識、同じツールを使いながら、立つ側が違う。

このナレッジベースで紹介するツールは、その両方に使える。

---

## なぜ存在を知ることに価値があるのか

「こういうツールがある」と知っているだけで、世界の見え方が変わる。

Webサイトを見たとき、「これはどうやって守られているんだろう」と考えるようになる。Wi-Fiに接続するとき、「このパスワードは安全なのか」と疑問を持つようになる。

使い方は、必要になったときにAIに聞けばいい。

でも、「そもそも何ができるのか」を知らなければ、質問すらできない。

---

## このナレッジベースの読み方

各ページは、ツールの「使い方」ではなく「物語」を語っている。

誰が作ったのか。なぜ生まれたのか。何ができるのか。

細かいコマンドやオプションは載せていない。それらは古くなる。でも、ツールが生まれた理由と、それが可能にすることは、時代を超えて価値がある。

さあ、始めよう。
